## 2025-03-13 - Semantic Documentation Diagrams
**Learning:** Using `<figure>` and `<figcaption>` for technical diagrams in documentation provides superior semantic grouping and accessibility compared to standard Markdown image syntax, allowing for explicit associations between the image and its description.
**Action:** Default to semantic HTML for architecture diagrams and complex technical visuals to improve the reading experience for both sighted and screen-reader users.
