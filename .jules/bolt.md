## 2026-02-21 - [Documentation Performance Optimization]
**Learning:** In repositories containing only documentation and images, performance gains can still be achieved by optimizing asset delivery and filename efficiency. Using HTML image tags with explicit dimensions, lazy loading, and async decoding improves Core Web Vitals (like CLS) and perceived load speed.
**Action:** Always check for unoptimized media and extremely long filenames in documentation-heavy repos. Use optimized HTML tags for images to enhance page performance.
