## 2025-05-14 - Documentation Performance Optimization
**Learning:** In documentation-heavy repositories, the documentation itself is the User Interface. UX and performance improvements should focus on making technical diagrams and text intuitive, accessible, and performant. Image loading efficiency (lazy loading, async decoding) and Layout Shift prevention (explicit width/height attributes) are critical for a fast documentation experience.
**Action:** Always optimize technical diagrams for web delivery by using modern formats like WebP, renaming assets to web-friendly filenames, and using HTML5 image attributes to optimize loading and prevent layout shifts.
